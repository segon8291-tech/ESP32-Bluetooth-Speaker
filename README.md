# ESP32-Bluetooth-Speaker
ESP32 Bluetooth Speaker. A2DP sink with buttons and LED visualizer. Pair as "Bluetooth Speaker". Connect GPIO25 (DAC) directly to audio amplifier input. Amplifier output to speaker. Buttons: Play/Pause (14), Next (4), Previous (13) to GND with pullups. LED on GPIO22 shows audio volume. Plays connection tones.
